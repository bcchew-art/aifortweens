A free AI assistant that lives on your own computer and chats through WhatsApp. It remembers your child, can search the web, and can be taught new abilities — called Skills. Think of it as your child's own personal Jarvis, running on your home machine instead of someone else's server.

What is Vibe Coding?

One of the most exciting skills your child will learn today. Here's how it compares to traditional coding:

Traditional Coding

  • 📖Learn syntax & logic first
  • ⏳Years to be productive
  • ✍️You write every line
  • 🐛One mistake breaks it
  • 🚧Barrier: technical knowledge

Vibe Coding

  • 💬Describe in plain English
  • ⚡Productive from day one
  • 🤖AI writes, you direct
  • 🔁Tell AI what's wrong, it fixes
  • 🎯Barrier: knowing what you want
💡 Key message: Vibe coding doesn't replace learning to code — but your child can build real things before they ever learn a single line of syntax. The skill being taught is thinking clearly and communicating precisely — valuable for life, not just tech.

AI Usage — What It Costs & How to Manage It

Every message to the agent uses a small amount of AI credit. Think of it like a prepaid data plan — when it's empty, you top it up. Here's how we suggest starting out:

✅ Stage 1 — Start Here
Prepaid AI Credits — $20
Load $20 of credit via OpenRouter. No monthly fee, no expiry. When it runs out, you top it up. Simple, transparent, and teaches kids to be thoughtful with how they use AI. $20 should last well beyond today's session.
⏳ Stage 2 — When Ready
ChatGPT Plus $30/mo + Prepaid as Backup
Once your child has good habits, switch to a flat monthly plan. Keep prepaid credits as a backup for when the monthly limit runs out. Only make this move when you're confident they won't burn through it.
⚠️ Why not ChatGPT Plus from day one? It has an unpredictable weekly usage cap. When the cap is hit, the agent goes completely silent for the rest of the week — with no warning. That's confusing for kids and frustrating for parents.

What NOT to Do — Important Lessons

The agent is powerful. That's exactly why it needs to be used responsibly. These aren't just rules — they're real concepts that even adult developers get wrong.

🔑Never save secret keys in plain text

When the agent connects to services like OpenRouter or Brave Search, it uses a secret API key. Never write it in a WhatsApp message, a notes app, or a text file on your desktop. If someone finds it, they can use your account and spend your credits.

💡 Rule: API keys live only in the agent's config file — nowhere else.
🏠Don't connect the agent to smart home devices without understanding the risk

OpenClaw can be given Skills to control things like Google Home, smart plugs, or door locks. That's exciting — but if someone sends a WhatsApp message to your agent pretending to be you, or if the agent misunderstands an instruction, it could unlock your front door, turn off your alarm, or switch off your router. This isn't sci-fi — it's how these systems work.

💡 Rule: Never give the agent control of anything physical without a parent reviewing the Skill first.
📁Remember: the agent can see everything on the computer

Once OpenClaw is installed, it has access to all files and folders on that machine. Don't install it on a computer that has sensitive family documents, banking files, or private photos. Use a dedicated machine or a clean old laptop — which is exactly what we're doing today.

💡 Rule: The agent's computer = the agent's computer. Keep it separate.
🤥Don't trust the agent blindly

AI can be confidently wrong. It might give you an answer that sounds completely correct but isn't. This is called a hallucination. For fun things like chess — no big deal. For anything that matters (medical, legal, financial) — always double-check with a real source.

💡 Rule: AI is a starting point, not the final word.
