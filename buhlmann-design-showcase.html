Consolidation Rules
Merge Join by Part No.
Match rows across Stock, Orders and Price files using Part Number.
Calc Available Qty
Available = On Hand - Allocated. Flag negatives as backorder.
Calc Inventory Value
Value = On Hand x Unit Cost (SGD). Roll up by category.
Flag Reorder Alert
Flag items where Available Qty falls below MOQ threshold.
Map Grade & Std
Preserve material grade (ASTM/DIN/EN/API) for traceability.
Merge Price Sync
Pull latest unit cost. Flag variance >5% vs last purchase.
1
Rule Cards Grid
3-column grid layout. Each card has a 3px left border that turns red on hover. Cards at 45% opacity when unchecked.
2
Rule Type Tags
Four categories: Merge (blue), Calc (yellow), Flag (red), Map (green). 8px uppercase text with matching tints.
3
Checkbox Styling
Native checkbox with accent-color: #c8102e. Clicking anywhere on the card label toggles the rule on/off.
4
Action Buttons
Secondary "Back" (outlined) and primary "Run Consolidation" (red fill). Right-aligned row with 8px gap.
